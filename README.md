## 🇺🇸 English
### Overview
**aladinIPTV Pro TV** is a professional-grade IPTV solution built with Flutter, specifically engineered for the Android TV ecosystem. It features a high-performance native Media3 (ExoPlayer) core with specialized stability fixes for hardware decoders.

### Key Features
*   **TV-First UI:** Full D-pad (Remote Control) navigation with Material 3 design.
*   **Native Player Performance:** Powered by Android Media3 (ExoPlayer) for ultra-low latency.
*   **Safe Zapping Technology:** Hard-reset logic for Realtek & lower-end SOCs to prevent system reboots.
*   **8 Multi-Language Support:** Auto-detection and manual switching.
*   **Smart Categorization:** Separate sections for Live TV, Movies, Series, and Favorites.
*   **EPG Engine:** Real-time electronic program guide integration.
